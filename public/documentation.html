A modern REST API for racing movies.

From Rush to Talladega Nights, this API allows users to find information about their favorite racing movies and add them to a personal list of favorites.

View the code »

Movies

HTTP Method Logic URL Request Format Response Format
GET Return a list of all movies /movies None Array of JSON objects with info about all movies
GET Return data about a single movie /movies/{title} None JSON object with info about a single movie
GET Return all movies of a specific genre /movies/genres/{genre} None Array of JSON objects with data about all movies of a specified genre
GET Return data about a single director /movies/directors/{director} None JSON object with data about a single director

Users

HTTP Method Logic URL Request Format Response Format
POST Add a new user /users JSON JSON object with data about added user
PUT Update data of a single user /users/{username} JSON JSON object with updates user data
POST Add a movie to user's list of favorites /users/{username}/favorites/{MovieID} None Text message confirming movie was successfully added to user's list of favorites
DELETE Remove a movie from user's list of favorites /users/{username}/favorites/{MovieID} None Text message confirming movie was successfully removed from user's list of favorites
DELETE Remove a user's account /users/{username} None Text message confirming the successful removal of user's account
