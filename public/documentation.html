fastFlix API Documentation

A modern RESTful API for racing films.

Movies
HTTP Method URL Logic Request Format Response Format
GET /movies Return a list of all movies None Array containing JSON objects with data of all movies
GET /movies/{title} Return data about a single movie None JSON object with data of a single movie
GET /movies/genres/{genre} Return all movies containing a specific genre None Array of JSON objects with data of all movies that contain specified genre
GET /movies/directors/{director} Return data about a single director None JSON object with data of a single director
Users
HTTP Method URL Logic Request Format Response Format
POST /users Add a new user JSON JSON object with data of added user
PUT /users/{username} Update data of a single user JSON JSON object with updated data of user
POST /users/{username}/favorites/{title} Add a movie to user's list of favorites None A text message confirming the title was successfully added to user's list of favorites
DELETE /users/{username}/favorites/{title} Remove single movie from user's list of favorites None A text message confirming the title was successfully removed from user's list of favorites
DELETE /users/{username} Remove a user's account None A text message confirming that the user's account was successfully removed
