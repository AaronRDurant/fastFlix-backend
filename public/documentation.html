fastFlix API Documentation

A modern RESTful API for racing films.

Movies
HTTP Method Logic URL Request Format Response Format
GET Return a list of all movies /movies None Array of JSON objects with info about all movies
GET Return data about a single movie /movies/{title} None JSON object with info about a single movie
GET Return all movies of a specific genre /movies/genres/{genre} None Array of JSON objects with data about all movies of a specified genre
GET Return data about a single director /movies/directors/{director} None JSON object with data about a single director

Sample Requests for Movie Data

GET all movies: /movies

GET data about a movie by title: /movies/Rush

GET data about a genre by title: /movies/genres/action

GET data about a director by name: /movies/directors/justin%20lin

Users
HTTP Method Logic URL Request Format Response Format
POST Add a new user /users JSON JSON object with data about added user
PUT Update data of a single user /users/{username} JSON JSON object with updates user data
POST Add a movie to user's list of favorites /users/{username}/movies/{MovieID} None Text message confirming movie was successfully added to user's list of favorites
DELETE Remove a movie from user's list of favorites /users/{username}/movies/{MovieID} None Text message confirming movie was successfully removed from user's list of favorites
DELETE Remove a user's account /users/{username} None Text message confirming the successful removal of user's account

Sample Requests for User Data

POST a new user: /users

PUT updates to data of a single user: /users/meganator

POST a movie to a user's list of favorites: /users/meganator/movies/{MovieID}

DELETE a movie from a user's list of favorites: /users/meganator/movies/{MovieID}

DELETE a user: /users/meganator
